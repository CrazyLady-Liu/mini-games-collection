# 悠游拼方 - 3x3 滑块拼图游戏

## 项目概述
- **项目名称**: 悠游拼方
- **项目类型**: 休闲益智拼图游戏
- **核心功能**: 3×3滑块拼图，通过滑动拼图碎片使数字/图案按顺序排列
- **目标用户**: 休闲游戏玩家

## 视觉与渲染规格

### 场景设置
- **布局**: 3×3 九宫格，8个数字/色块 + 1个空白格
- **背景**: 柔和的渐变背景，淡雅的马卡龙色系
- **画布尺寸**: 响应式，适应移动端和桌面端

### 材质与效果
- **拼图碎片**:
  - 马卡龙色系（薄荷绿、樱花粉、淡黄色、薰衣草紫、蜜桃色等）
  - 纯色或带数字标记
  - 圆角设计，柔和阴影
- **空白格**: 略深于背景的凹槽效果
- **动画效果**:
  - 碎片滑动: 0.2s ease-out 过渡
  - 通关成功: 缩放+透明度动画，柔和的粒子效果
  - 难度切换: 淡入淡出过渡

## 游戏逻辑规格

### 拼图生成
- 初始状态: 1-8按顺序排列，空白格在右下角
- 打乱算法: 随机执行N次合法滑动（保证可还原）
- 打乱次数: 50-100次随机合法移动

### 交互控制
- **点击滑动**: 点击空白格相邻的碎片，自动滑动
- **键盘支持**: 方向键控制空白格周围的碎片移动
- **动画时长**: 200ms ease-out

### 通关判定
- 当碎片顺序为 [1,2,3,4,5,6,7,8,空] 时判定通关
- 通关触发柔和成功动画（1.5秒）
- 自动进入下一难度（暂时仅一个难度级别）

### 功能按钮
- **重置本局**: 恢复到当前打乱后的初始状态
- **重新打乱**: 生成新的随机拼图

## 技术实现

### 文件结构
```
悠游拼方/
├── index.html      # 主HTML文件
├── style.css       # 样式文件
└── script.js       # 游戏逻辑
```

### 状态管理
```javascript
{
  tiles: [1,2,3,4,5,6,7,8,0],  // 0表示空白格
  emptyIndex: 8,              // 空白格位置
  initialTiles: [...],        // 打乱后的初始状态（用于重置）
  moveCount: 0,               // 移动次数
  isAnimating: false,         // 动画锁
  difficulty: 1               // 当前难度
}
```

### 核心函数
- `initGame()`: 初始化游戏
- `shuffle()`: 打乱拼图（保证可还原）
- `canMove(index)`: 判断是否可以移动
- `moveTile(index)`: 移动碎片
- `checkWin()`: 检查通关
- `resetGame()`: 重置本局
- `newGame()`: 重新打乱
